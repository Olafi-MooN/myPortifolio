01. API Rest

Envia ou recebe uma estrutura de dados

Métodos HTTP
  • GET: Buscar informações

    http://localhost:3000/users

  • POST: Criar uma informação.

    http://localhost:3000/users

  • PUT/PATH: Alterar uma informação.

    http://localhost:3000/users/1

  • DELETE: Deletar informações.

    http://localhost:3000/users/1

JSON (Javascript object notation)

Estrutura de objetos global, para facilitar a comunicação entre o front e back-end

{
"users":{
					"name": "Fulano"
					"tech": "ReactJS, NodeJS, Javascript "
				} 
}

Conteúdo da requisição

Vermelho = "Route";

Amarelo = "Route params";

Azul = "Query params"

htttp://localhost:3000/company/1/users/page=2
