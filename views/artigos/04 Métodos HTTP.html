04. Métodos HTTP

GET: Buscar informações

Criando uma rota que busca uma informação:

const express = require('express');
const server = express();

server.use(express.json());

server.get('/projects',(req, res) =>  res.json({ list: ["produto1", "produto2"] }));

server.listen(3000, () => 'Servidor iniciado');

A gente pode acessar esse recurso de project pelo navegador usando: http://localhost:3000/projects

POST: Criar uma informação.

Criando uma rota/recurso que cria informações

const express = require('express');
const server = express();

server.post('/projects', (req, res) => res.json(
    { list: ["produto1", "produto2", "produto3"] }
));

server.listen(3000, () => console.log('^_^ Server Start'));

Por padrão o navegador só exibe métodos do tipo GET, para visualizar outros tipos de métodos é necessário utilizar o Insomia.

PUT/PATCH: Alterar uma informação.

Utilizando o put para alterar uma ou varias informações, é interessante relembrar que as rotas do tipo put geralmente utilizam um ":id" que identifica oque vai ser alterado 

const express = require('express');
const server = express();

server.put('/projects/:id', (req, res) => res.json(
    { list: ["produto2", "produto1", "produto3"] }
))

server.listen(3000, () => console.log('^_^ Server Start'));

Por padrão o navegador só exibe métodos do tipo GET, para visualizar outros tipos de métodos é necessário utilizar o Insomia.

DELETE: Deletar informações.

Por fim vamos ver como deletar uma rota/recurso:

const express = require('express');
const server = express();

server.delete('/projects/:id', (req, res) => res.json(
    { list: ["produto2", "produto1"] }
));

server.listen(3000, () => console.log('^_^ Server Start'));

Mostrando o código completo

const express = require('express');
const server = express();

server.get('/projects',(req, res) => res.json(
    { list: ["produto1", "produto2"] }
));

server.post('/projects', (req, res) => res.json(
    { list: ["produto1", "produto2", "produto3"] }
));

server.put('/projects/:id', (req, res) => res.json(
    { list: ["produto2", "produto1", "produto3"] }
));

server.delete('/projects/:id', (req, res) => res.json(
    { list: ["produto2", "produto1"] }
));

server.listen(3000, () => console.log('^_^ Server Start'));
