07. Middlewares

São os interceptadores do express, podem interromper ou alterar dados de uma requisição

Um middlewares nada mais é do que uma função do javascript que recebe 3 parâmetros ( request, response e next). Um exemplo de middlewares são as rotas, mas as rotas não necessitam de um next.

Os middlewares são usados quando se quer disparar trechos de códigos em uma ou mais rotas...

Oque vamos fazer agora é um middleware que visualiza qual é a rota e o método, que esta sendo chamando, Veja o exemplo;

const express = require('express');
const server = express();

// Criando a função middle
function middlewares( request, response, next){
	const { method, url } = request; // pegando o method e a url do request
	const logLabel = `${method.toUpperCase()} ${url}`;
	
	console.log(logLabel);
}

// É  necessario declarar que queremos usar o middleware, esta decalaração faz com que
// Todas as rotas utilizem esse middle
server.use(middlewares); 

server.ger('/projects', (req, res) => {
	res.json('middlewares');
});

Executando o código acima você notara que as rotas/recursos não será carregada, isso porque neste caso ele interrompeu totalmente a requisição, poderíamos corrigir isso, utilizando o next(), quando acabarmos de realizar tudo oque o middleware necessitava fazer. Veja o exemplo:

// Criando a função middle
function middlewares( request, response, next){
	const { method, url } = request; // pegando o method e a url do request
	const logLabel = `${method.toUpperCase()} ${url}`;
	
	console.log(logLabel);
	return next();
}

Como usar o middleware em apenas uma rota/recurso especifica:

Existe duas maneira de aplicar o middleware em uma rota especifica, um exemplo seria chamando a função na própria rota.

//Colocamos a função dentro da função server.get()
server.get('/projects',  middlewares, (req, res) =>{
	res.json('Ok')
})

Também pode ser chamada varias funções

//Colocamos a função dentro da função server.get()
server.get('/projects',  middlewares,  middlewares2,  middlewares3, (req, res) =>{
	res.json('Ok');
});

Uma outra maneira é usar:

É utilizando o express.use() e definindo a rota que se quer aplicar a função, também podemos adicionar uma ou varias funções:

server.use('/projects/:id', middlewares,  middlewares2,  middlewares3);
