06. Aplicação funcional nodeJS

Com os conceitos anteriores, da para fazer um pequeno CRUD

Abra o terminal na pasta que deseja iniciar o seu projeto e rode os seguinte comandos:

Para iniciar o package.json

npm init -y

Vamos instalar as dependências do projeto:

  • uuidv4 é um gerador de id único.
  • express que é o nosso gerenciador de rotas.

npm i uuidv4
npm i express

Com essas dependências instaladas crie um um arquivo index.js e digite o seguinte código:

// Adicionando as dependêcias que instalamos dentro do arquivo index.js
const express = require('express');
const { uuid } = require('uuidv4');

const server = express();

Com isso temos todas dependências que necessitamos dentro do projeto, o próximo passo é configurar as rotas/recursos da aplicação. Mas antes disso temos que criar uma variável local para guardar os nossos projetos ja que a ainda não estamos trabalhando com um banco de dados.

var projects = [];

Também vamos trabalhar com o envio e dados JSON, para permitir esse acesso no express, precisamos de um midleware que tratara esses dados.

server.user(express.json());

A primeira rota a ser criada, será a que exibira os projetos.

server.get('/projects', (req, res) => {
	// Antes de mostrar os projetos queremos saber se a nossa rota possui uma Query Params 
    const { title } = req.query;
	// Vamos atribuir o resultado da condição para dentro da const result
    const result = title // Verifica se o title é diferente de nulo
        ? projects.filter(project => project.title.includes(title))
        : projects;
    
    res.json(result) // mostramos o resultado da condição   
});

Agora vamos adicionar a rota que criara novos projetos.

server.post('/projects', (req, res) => {
	// Aqui é recebido da rota/recurso um Request Params 
    const { title, name } = req.body;
		
		// Cria-se um projeto com os valores recebidos pela requisição
    const project = { id: uuid(), title, name };
		// Sera adicionado esse projeto recem criado para dentro da nossa lista de projects
    projects.push(project);

    res.json(project)
});

Agora vamos criar uma rota para atualizar um projeto já criado pelo valor do seu id.

// Logo no começo do código percebemos algo diferente, a nossa rota agora tem um :id
// Esse id é resposavel por nós indicar qual projeto será ALTERADO
server.put('/projects/:id', (req, res) => {
    const { id } = req.params; // Vamos receber o id da nossa rota/recurso
    const { title, name } = req.body; // Vamos receber os valores que foram alterados
		
		// Procurando pelo projeto que tem o mesmo id recebido pela rota
    const projectIndex = projects.findIndex(project => project.id === id);
    if(projectIndex > 0) // Caso esse id não seja encontrado dentro do projeto
        res.status(400).json(' Project not found! ');

    const project = {title, name, id}; // Criando o novo projeto atualizado
    projects[projectIndex] = project; // Adicionadno esse projeto novamente aos projects 

    res.json( project )
});

Agora vamos criar uma rota, para deletar um projeto já criado, pelo valor do seu id.

// Logo no começo do código percebemos algo diferente, a nossa rota agora tem um :id
// Esse id é resposavel por nós indicar qual projeto será DELETADO

server.delete('/projects/:id', (req, res) => {
    const { id } = req.params; // Vamos receber o id da nossa rota/recurso
		
		//Verificando se o projeto realmente existe
    const projectIndex = projects.findIndex(project => project.id === id);
    if(projectIndex > 0) 
        res.status(400).json(' Project not found! ');
    
		// Deletando esse projeto
    projects.splice(projectIndex, 1);

    res.status(200).send( ` Removido com sucesso!` )
});

Por fim falta apenas a configuração da porta que a nossa aplicação estará rodando.

server.listen(3000, () => console.log('^_^ Server Start'));
