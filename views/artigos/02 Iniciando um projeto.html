02. Iniciando um projeto

Para iniciar um projeto utilizando o node.JS é muito simples

Abra o terminal em uma pasta que deseja iniciar o projeto e adicione o seguinte comando:

npm init -y

Este comando basicamente criara o package.json que ajudara a gerenciar as dependências do projeto. o -y serve para aceitar automaticamente a configuração, não é obrigatório usar o -y.

Ao realizar a instalação será exibido um Json com a seguinte estrutura;

{
  "name": "nome-do-projeto",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "author": "",
  "license": "ISC"
}

Pronto oque podemos fazer agora é criar um pasta "src", para uma melhor organização do projeto. Dentro dessa pasta vamos criar um "index.js";

>src

>>index.js

Agora vamos instalar um micro framework gerenciador de rotas chamado "express", para que esse gerencie a nossa aplicação:

npm install express

com "express" instalado abra o arquivo "index.js" que foi criado anteriormente, e adicione as seguintes linhas de code;

// Buscando pelo express instalado 
const express = require('express');

// Atibuindo os metodos de express para dentro da constante serve
const server = express();

// Como estamos mandando é recebendo dados JSON, precisamos criar um middleware para interceptar esses dados
server.use(express.json()); 

// Enviando a mensagem para o navegador
server.get('/', (req, res) => res.json({ message: 'Olá Mundo' }));

// Acessa os dados pela porta 3000
server.listen(3000);

Feito o isso o seu primeiro projeto ja esta criado para executa-ló abra o terminal e digite:

node src/index.js
